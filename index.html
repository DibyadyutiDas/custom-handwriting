API Integration

Integrate our AI handwriting conversion into your own applications with our simple, powerful API.

Making API Requests

Send a POST request to our endpoint with your text and styling preferences.

javascript
fetch('https://api.writeai.com/v1/handwriting', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_API_KEY'
  },
  body: JSON.stringify({
    text: 'Hello, this is my handwriting!',
    style: 'casual',
    format: 'png' // or 'pdf', 'svg'
  })
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));

Response Format

The API returns a JSON response with the URL to your generated handwriting image.

json
{
  "success": true,
  "id": "hw_12345abcde",
  "url": "https://cdn.writeai.com/handwriting/hw_12345abcde.png",
  "text": "Hello, this is my handwriting!",
  "style": "casual",
  "format": "png",
  "created": "2025-04-15T10:30:45Z"
}

Node.js Example

Here's a complete Node.js example using axios to interact with our API.

javascript
const axios = require('axios');

const generateHandwriting = async () => {
  try {
    const response = await axios.post('https://api.writeai.com/v1/handwriting', {
      text: 'Hello, this is my handwriting!',
      style: 'casual',
      format: 'png'
    }, {
      headers: {
        'Content-Type': 'application/json',
        'Authorization': 'Bearer YOUR_API_KEY'
      }
    });
    
    console.log(response.data);
    return response.data;
  } catch (error) {
    console.error('Error generating handwriting:', error);
  }
};

generateHandwriting();
